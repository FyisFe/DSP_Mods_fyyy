# FastTinderLaunch

Restore old 100% dark fog tinder launch probability (launch when full).
将黑雾火种的发射概率恢复为旧版 100%（充满即发）。

- Remove 1800-tick launch interval / 移除 1800 tick 发射间隔
- Remove probability decay and global coverage dampening / 移除概率衰减和全局覆盖率抑制
- Remove build trigger phase restriction / 移除建造触发相位限制

## Dependencies / 依赖

- [BepInEx](https://github.com/BepInEx/BepInEx) 5.x
- [UXAssist](https://github.com/shabbywang/UXAssist)

## Configuration / 配置

Toggle on/off in UXAssist config panel (Alt+\`).
在 UXAssist 设置面板（Alt+\`）中勾选/取消 "Restore old 100% tinder launch behavior"，即时生效。
