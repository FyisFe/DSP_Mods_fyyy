# BuildingPosViewer

Adds x/y/z copy buttons to building inspection windows. Click a button to copy that coordinate value to clipboard.

为建筑物检查窗口添加x/y/z复制按钮。点击按钮即可将对应坐标值复制到剪贴板。

## Features / 功能

- Copy buttons appear next to the building title in inspection windows / 复制按钮显示在检查窗口的建筑标题旁
- Click x, y, or z to copy the respective grid coordinate (longitude, latitude, altitude) to clipboard / 点击x、y或z可将对应的网格坐标（经度、纬度、高度）复制到剪贴板
- Supports 18 building window types / 支持18种建筑窗口类型

## Install / 安装

Install via R2Modman or extract the DLL to `BepInEx/plugins/`.

通过R2Modman安装，或将DLL解压到 `BepInEx/plugins/`。

## Requirements / 依赖

- BepInEx 5.x
